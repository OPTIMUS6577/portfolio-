# 🚀 WORK HUB | Standalone Digital Services Platform

A high-fidelity digital engineering and creative alchemy hub.

## 🛠️ Features

- **PWA (Progressive Web App)**: Can be installed on mobile devices.
- **Telegram Integration**: Instant alerts for project inquiries.
- **Netlify Forms**: Automatic submission handling.
- **Admin Panel**: Accessible via `admin.html`.

## 🚀 Deployment

1. Drag and drop this folder to **[Netlify Drop](https://app.netlify.com/drop)**.
2. Or connect to a new GitHub repository for automated deployments.
