«Қашқадарё вилоятидаги Оғир ва Янги Ўзбекистон қиёфасидаги ҳудудлар платформаси» учун техник топшириқ. 
1. ЛОЙИҲА НОМИ
«Қашқадарё вилояти ҳудудларини манзилли дастур асосида комплекс мониторинг қилиш платформаси»
Мазкур техник топшириқ жараён давомида мукаммалаштириб борилади
(Қисқартирилган номи: HududMonitor)
2. ЛОЙИҲАНИНГ МАҚСАДИ
Вилоятда амалга оширилаётган “Оғир туман” ва “Янги Ўзбекистон қиёфасидаги туман/маҳалла” дастурлари доирасида ажратилган маблағлар, бажарилиши режалаштирилган ишлар ҳамда ҳудудларнинг ижтимоий-иқтисодий кўрсаткичларини бирлаштирган интерактив веб-платформани яратиш.
3. ФУНКЦИОНАЛ ТАЛАБЛАР
3.1. Асосий интерфейс тузилмаси
Интерфейс 3 колонкали бўлиши керак:
Чап панель
Марказий панель
Ўнг панель
Ҳудудлар рўйхати (туманлар + маҳаллалар)
Инфографикалар ва диаграммалар
Паспорт маълумотлари ва манзилли дастур
Экран кенглиги 1000 пикселдан кичик бўлганда, панеллар вертикал тартибда (бир-бирининг остида) жойлашиши керак.
3.2. Чап панель (рўйхат)
Чап панель икки асосий бўлимдан иборат:
а) Оғир туманлар
Касби тумани
Нишон тумани
Деҳқонобод тумани
б) Янги Ўзбекистон қиёфасидаги туманлар
Қамаши тумани
Яккабоғ тумани
в) Маҳаллалар (ҳар бир туманга тегишли камида ПҚ-298 ва ПҚ-49 та маҳалла)
Ҳар бир сатрда қуйидаги маълумотлар бўлиши керак:
Ҳудуд номи
Тури (Оғир / Янги Ўзб. / Маҳалла)
Иконка (мос равишда)
Функционаллик: Ҳар қандай элементни босганда (туман ёки маҳалла) марказий ва ўнг панеллардаги барча маълумотлар шу ҳудудга мос ҳолда янгиланиши керак.

8. МАЪЛУМОТЛАРНИ БОШҚАРИШ ВА КЕНГАЙТИРИШ
1-босқич: барча маълумотлар JavaScript ичида статик ҳолда сақланади.
2-босқич (истиқбол): маълумотлар API орқали сервердан олинадиган қилиб тузилган бўлиши керак (модулларга ажратилган).
Барча маълумотлар бир марказий объектда жамланган бўлиши, кейинги қўшиладиган ҳудудларни қўшиш осон бўлиши керак.

11. ТОПШИРИҚНИ ҚАБУЛ ҚИЛИШ МЕЗОНЛАРИ (DoD – Definition of Done)
1. Чап панелдаги ҳар бир элемент (туман ёки маҳалла) босилганда:
kpi карточкалари тўғри янгиланади;
4 хил диаграмма тўғри маълумотлар билан чизилади;
ўнг панелда ҳудуд паспорти ва манзилли дастур лойиҳалари тўлиқ кўрсатилади.
2. Тумондан маҳаллага ўтилганда диаграммалар маҳалла миқёсига мослашади.
3. Интерфейс браузер ўлчами ўзгарганда ҳам тўғри кўринади.
4. Браузер консолида хатоликлар мавжуд эмас.
5. Лойиҳа кодлари архив ёки Git репозиторий сифатида топширилади, код тоза ва изоҳланган.
